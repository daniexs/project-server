### POST /user/register
#### Description
- Create a new user

#### Request
- Body
    ```json
    {
        "username": "string",
        "email":"string",
        "password": "string",
    }
    ```
#### Response
_201 - Created_
- Body
    ```json
    {
      "id": "Id",
      "email": "String"
    }
    ```

_400 - Bad Request_
- Body
    ```json
    {
        "message": "Email must be unique"
    }
    {
        "message": "Email is required"
    }
    {
        "message": "Password is required"
    }
    {
        "message": "Invalid email format"
    }
    ```
### POST /user/login

_401 - Unauthorized_

    ```json
    }
    ```
